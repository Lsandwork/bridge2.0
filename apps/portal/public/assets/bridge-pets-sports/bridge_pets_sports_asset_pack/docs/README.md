
# Bridge Pets Sports Fan Gear Asset Pack

Production-ready original placeholder assets for the Bridge Pets Sports Partnership / Fan Gear feature.

## Important legal note
This pack intentionally contains **no real FIFA, NBA, MLB, NFL, Lakers, Dodgers, player, league, or team logos**. It is built as a licensing-ready asset system for Codex to install safely. Replace placeholders only after licensed assets are approved.

## Included
- PNG icons: `png/icons/`
- SVG vector icons: `svg/icons/`
- PNG accessories: `png/accessories/`
- SVG accessory vectors: `svg/accessories/`
- PNG pet sprites/states: `png/pets/`
- SVG pet base files: `svg/pets/`
- Backgrounds: `png/backgrounds/`, `svg/backgrounds/`
- Config seed JSON: `config/`
- PSD-compatible boards: `psd/`
- Preview boards: `preview_boards/`

## Recommended Codex install paths
Copy assets into your Bridge repo:

```bash
public/pets/fan-gear/icons
public/pets/fan-gear/accessories
public/pets/fan-gear/effects
public/pets/fan-gear/backgrounds
public/pets/companions
src/lib/pets/fanGear/seeds
```
